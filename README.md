# Banking Management System

A simple Banking Management System built using Python that allows users to perform basic banking operations like creating accounts, depositing money, withdrawing money, and checking account balance. This project uses core Python concepts and is ideal for beginners.

## Features
- Create a new savings account
- Auto-generate unique account numbers
- Deposit money into an account
- Withdraw money with minimum balance rule (₹500)
- Check account balance with yearly interest calculation (7.25%)
- Display all account details

## Technologies Used
- Python
- Data Structures (Dictionary, List, Set)

## How It Works
The system runs in a loop and provides a menu-driven interface where users can select different banking operations. It ensures unique account numbers, maintains a minimum balance, and calculates interest for savings accounts.

## How to Run
1. Download or clone this repository
2. Open terminal or command prompt
3. Run the file using:
   python Banking_Management_System.py
